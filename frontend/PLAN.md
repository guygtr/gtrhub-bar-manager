# Frontend Plan for GTRHub Bar Manager

## Overview
- Built with React using create-react-app.
- Features: Stock Tracking and Recipe Management.
- Navigation via react-router-dom.
- Basic components with local state (useState).
- Assumed designs: Simple lists and forms.
- Assumed architecture: Standalone frontend, mock data (integrate with backend later).

## Structure
- App.js: Router setup with routes for /stock and /recipes.
- pages/StockTracker.js: Displays stock list, add form.
- pages/RecipeManager.js: Displays recipe list, add form.

## Next Steps
- Integrate with backend APIs from @Mario's architecture.
- Apply UI designs from @Ezio (wireframes, styles).
- Add features like editing, deleting, stock checks for recipes.
